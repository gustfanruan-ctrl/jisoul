# JiSoul RAG 高精度检索完整改造清单

## 改造目标

解决两个核心问题：
1. **Re-ranking 精排缺失** → 添加 Cross-Encoder 重排序
2. **JSON 标签未利用** → 关键词匹配 + 优先级加权 + 意图精准过滤

## 重建向量索引（关键！）

Embedding 模型从 `bge-base-zh-v1.5` 升级到 `bge-large-zh-v1.5`，**必须重建向量索引**：

```bash
cd jisoul/backend

# Step 1: 清空旧向量数据
rm -rf data/chroma/*

# Step 2: 重新导入知识库
# 方式A: 使用现有的 upload.py
python upload.py

# 方式B: 使用 seed.py（如有）
python -m app.knowledge.seed --rebuild

# Step 3: 确认导入成功
# 检查向量库条目数
ls -la data/chroma/
```

## 配置参数说明

在 `config.py` 中可调整：

```python
# 是否启用 rerank（测试时可关闭对比）
RERANK_ENABLED: bool = True

# Embedding 模型（可回退到 base）
EMBEDDING_MODEL_NAME: str = "BAAI/bge-large-zh-v1.5"

# Reranker 模型（可用更小的）
RERANKER_MODEL_NAME: str = "BAAI/bge-reranker-base"

# 粗排候选数（影响 rerank 计算量）
RAG_TOP_K: int = 15

# 相似度阈值（粗排放宽）
RAG_SIMILARITY_THRESHOLD: float = 0.35

# Re-ranking 触发阈值
RERANK_SCORE_SPREAD_THRESHOLD: float = 0.08
RERANK_LOW_CONFIDENCE_THRESHOLD: float = 0.55
```

```bash

```
